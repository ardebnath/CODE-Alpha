# Alpha Kids Studio

This folder contains a kids-friendly browser studio for the Alpha language.

What it includes:

- colorful lesson cards
- simple project builders
- voice help through the browser speech system
- friendly error explanations
- a bright Alpha editor that runs real Alpha code

Run it with:

```powershell
python "kids frendly alpha/kids_server.py"
```

Open:

```text
http://127.0.0.1:8090
```

To share it on the same trusted Wi-Fi:

```powershell
python "kids frendly alpha/kids_server.py" --share-lan
```
