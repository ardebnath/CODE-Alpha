## Problem Solved

Describe what Alpha solves:
Alpha solves the gap between “easy for humans to read” and “powerful enough to build real programs.”

Alpha solves these problems:

1.Hard-to-read code: Alpha uses simple words like note, set, repeat, and function.
2.Beginner confusion: it is easier to learn than low-level syntax-heavy languages.
3.Slow idea testing: Alpha is interpreted, so you write and run immediately.
4.Mixing human readability with real power: it still supports loops, functions, files, database, packages, and graphics.
5.Weak small-project tools: Alpha has built-in SQLite storage, package support, and a browser studio.
6.Device access: you can write code from PC and mobile through Alpha Studio.
7.Safe extension: you can add more keywords, modules, bridges, and packages without changing the whole language
