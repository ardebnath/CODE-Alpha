# Alpha Mobile Coding App

This folder contains a phone-first Alpha coding app built as a mobile-friendly browser PWA.

What it includes:

- a mobile-first Alpha editor
- touch-friendly snippet buttons
- local notebook saves on the current device
- featured sample programs
- output and translated Python views
- installable app shell files for browser app mode

Run it with:

```powershell
python "alpha mobile coding app/mobile_server.py"
```

Open:

```text
http://127.0.0.1:8100
```

To open it on your phone using the same trusted Wi-Fi:

```powershell
python "alpha mobile coding app/mobile_server.py" --share-lan
```

Then open the printed `192.168...` address on your phone.
